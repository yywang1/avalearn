seajs学习笔记

打包

spm build

本地部署到sea-modules目录(安装模板自身)

spm install . -d sea-modules

怎样加载已被打包到main.js中的某模块

//例：模板中使用abTest
seajs.use(['jquery', 'main'], function ($, main) {
	var ABTest = seajs.require('custom/main/0.0.1/mod/abTest');
	var abTest = new ABTest();
	var colorSelectVersion = abTest.getVersion('ColorSelect');
	if (colorSelectVersion == 'new') {
		//do something...
	}
});

PS：模块标识id和路径相匹配，可在打包后的main.js中找到如下代码

define("custom/main/0.0.1/mod/abTest", [...], function () { ... })
