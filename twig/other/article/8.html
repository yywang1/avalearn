Sublime 使用笔记

Sublime Text2安装Package Control

方法一：

访问 Package Control 站点按照提示复制一段python命令安装Package Control

  • 1. ctrl+` 或 View > Show Console 打开Sublime控制台
  • 2. 复制相应版本的代码，粘贴到Sublime控制台中。
    下面是 Sublime Text2 的代码 
    import urllib2,os; pf='Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler( ))); open( os.path.join( ipp, pf), 'wb' ).write( urllib2.urlopen( 'http://sublime.wbond.net/' +pf.replace( ' ','%20' )).read()); print( 'Please restart Sublime Text to finish installation')
  • 3. 回车，OK

方法二：

下载Package Control插件包 Package Control.sublime-package 到插件目录 Preference -> Browse Packages，重启Sublime

插件安装：

  • 1. ctr+shift+P 调出控制面板
  • 2. 输入 Install Package
  • 3. 输入并选择你要安装的package
  • 4. 回车

插件卸载：

  • 1. ctr+shift+P 调出控制面板
  • 2. 输入 Remove Package
  • 3. 输入你要删除的package
  • 4. 回车，OK

列出全部扩展

  • 1. ctr+shift+P 调出控制面板
  • 2. 输入 List Packages
  • 3. 回车

安装CSS格式化插件CompactExpandCss

  • 1. ctr+shift+P -> 输入Install Package -> 输入并选择CompactExpandCss
  • 2. 选择Preferences -> Key Bindings - User，添加如下代码
    { "keys": ["ctrl+super+]"], "command": "compact_expand_css", "args": { "action": "expand" }  },
{ "keys": ["ctrl+super+["], "command": "compact_expand_css", "args": { "action": "compact" } }
  • 3. 打开一个css文件，Ctrl+Alt+[ 格式化为一行显示，Ctrl+Alt+] 格式化为多行显示。

ps：站长工具在线CSS格式化：http://tool.chinaz.com/Tools/CssFormat.aspx，使用这个可以统一空格。
