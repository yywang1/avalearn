常用CSS代码

Chrome中文版不支持12px以下的字体大小（英文版没有限制）：

html { -webkit-text-size-adjust: none; }  /*webkit的私有属性*/

PC桌面版Chrome 27正式取消了-webkit-text-size-adjust属性的支持，如果定义该属性在Chrome调试窗口会显示Unknown property name，所有字号最小为12px。但是，移动端chrome/safari目前依然支持-webkit-text-size-adjust属性。

.chrome_adjust {
font-size: 9px;
-webkit-transform: scale(0.75);
-o-transform: scale(1);    //针对能识别-webkit的opera browser设置
}

针对Chrome 27以后版本的浏览器，可以通过上面的方法进行局部缩放

有序列表，数字排序

ol { list-style: decimal; margin-left: 40px; }
ol li { list-style: decimal-leading-zero; list-style-position: outside; }

注：属性为 decimal 时，数字编号从1,2,3开始；属性为 decimal-leading-zero 时，数字编号从01,02,03开始。

      <ol start="2"> start 属性规定有序列表的开始点。

微软雅黑

font-family: Microsoft YaHei;

火狐强制出现滚动条

html {
overflow: -moz-scrollbars-vertical;
}

display:inline-block 属性

display: inline-block;
*display: inline;
zoom: 1;

外层元素

font-size: 0;
*letter-spacing: -1px;

css透明属性

filter: alpha(opacity=40);
opacity: 0.4;

强制大小写

text-transform: uppercase;   /*全大写*/
text-transform: lowercase;   /*全小写*/
text-transform: capitalize;    /*首字母大写*/

元素没有背景css

background: transparent;

Clearfix Hack

.clearfix:after { content:"."; display:block; height:0; clear:both; visibility:hidden; }
.clearfix { display:inline-block; }
.clearfix {display:block;}

文字超出时省略号表示

overflow: hidden;
white-space: nowrap; 
text-overflow: ellipsis;

全是字母或数据组成的字符串强制换行，空格正常换行

td {
word-wrap: break-word;
}
