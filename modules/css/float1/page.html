浮动层

效果展示

[关闭]

代码展示

HTML

<!--灰背景-->
<div class="bg" style="display:none;">
    <iframe src="" style="width:100%; height:100%; top:0px; left:0px; position:absolute; visibility:inherit; z-index:-1; filter:alpha(opacity=0); opacity:0; bacKground:#0F9; " frameborder=0></iframe>
</div>
<!--浮动层-->
<div class="floatDiv" style="display:none;">
    <a href="javascript:void(0)" class="close"> [关闭] </a>
</div>

CSS

/*灰背景*/
.bg {
    width:100%;
    height:100%;
    _width:expression(document.body.clientWidth);
    _height:expression(document.body.clientHeight);
    left:0;
    top:0;
    background:#666;
    filter:alpha(opacity=30);
    opacity:0.3;
    z-index:9000;
    position:fixed !important;
    position:absolute;
}

/*浮动层*/
.floatDiv {
    left:50%;
    top:50%;
    width:300px;
    height:200px;
    margin-left:-150px !important;
    margin-top:-100px !important;
    margin-top:0px;
    position:fixed !important;/*FF IE7*/
    position:absolute;/*IE6*/
    z-index:9999;
    _top: expression(eval(document.compatMode && document.compatMode=='CSS1Compat') ? documentElement.scrollTop + (document.documentElement.clientHeight-this.offsetHeight)/2 : document.body.scrollTop + (document.body.clientHeight - this.clientHeight)/2);
    background:#397300;
    overflow:hidden;
}

/*关闭按钮*/
.close {
    display:block;
    width:100px;
    height:30px;
    line-height:30px;
    margin:85px auto;
    text-align:center;
    background:#fff;
}

JS

<script type="text/javascript" src="../../../jquery-1.6.1.min.js"></script>
<script>
    $(document).ready(function(){
        $(".btnShow").click(function(){
            $(".bg").show();
            $(".floatDiv").show();
        });
        $(".close").click(function(){
            $(".bg").hide();
            $(".floatDiv").hide();
        });
    })
</script>

要点总结

  • 当浮动代码外层标签中存在position:relative;或position:absolute;时容易引起bug，建议该段代码放在body的最后面。
  • IE6中select层级过高，会盖在浮动层上面，针对这一问题，在灰背景中添加iframe标签可解决。
