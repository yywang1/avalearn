CSS

/*一级菜单*/
.topnav { background:#397300; width:100%; height: 34px; float:left; }
.topnav .li1 { display:block; float: left; height: 34px; }
.topnav .a1 { display: block; color:#fff; height: 34px; line-height: 34px; float: left; white-space:nowrap; padding: 0 36px 0 35px; }
.topnav .a1:hover { color:#fff; background:#285000;}
/*二级菜单*/
.topnav, .topnav ul, .topnav li, .topnav a { overflow:visible }
.topnav li {position:relative; z-index:500; vertical-align:bottom; }
.topnav a { display:block; _overflow:hidden; }
.topnav a:hover { border-bottom:0; overflow:visible }/*must have this*/
.topnav table { _margin-top:-5px; }
.topnav td { padding:0; margin:0; }
.topnav .ul2 { visibility:hidden; position:absolute; left:0px; top:34px; z-index:1000; width:190px; background:#3e7b00; border:solid 1px #285000; border-bottom:none;}
.topnav .li1:hover .ul2, .topnav .a1:hover .ul2 { visibility:visible; }
.topnav .li2 {width:100%; border-bottom:solid 1px #285000;}
.topnav .a2 {padding:0 10px; line-height:25px; color:#FFF;}
.topnav .a2:hover { background:#285000; color:#FFF;}
/*三级菜单*/
.topnav .ul3 { visibility:hidden; position:absolute; left:190px; top:-1px; z-index:1000; width:190px; background:#3e7b00; border:solid 1px #285000; border-bottom:none;}
.topnav .li2:hover .ul3, .topnav .a2:hover .ul3 { visibility:visible; }
.topnav .li3 {width:100%; border-bottom:solid 1px #285000;}
.topnav .a3 {padding:0 10px; line-height:25px; color:#FFF;}
.topnav .a3:hover { background:#285000; color:#FFF;}

要点总结

  • overflow属性是关键，建议该段代码中除了现存的overflow属性以外不要出现overflow:hidden。
  • 如果鼠标无法悬浮在子菜单上，可以尝试给它的父标签设置高度或宽度。
