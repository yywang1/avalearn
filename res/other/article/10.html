git 学习笔记（四）—— difftool

打开 .gitconfig 文件

$ ~/.gitconfig

添加以下代码

[diff]
tool = sourcetree
[difftool]
prompt = false
[difftool "sourcetree"]
cmd = \"D:/Program Files (x86)/Beyond Compare 3/BCompare.exe\" \"$LOCAL\" \"$REMOTE\"
[merge]
tool = sourcetree
[mergetool]
prompt = false
[mergetool "sourcetree"]
trustExitCode = true
cmd = 'D:/Program Files (x86)/Beyond Compare 3/BCompare.exe' \"$LOCAL\" \"$REMOTE\" \"$BASE\" \"$MERGED\"

命令行中使用

$ git difftool [filename]

遇到问题：使用免安装的Beyond Compare不起左右，使用安装版才行。
