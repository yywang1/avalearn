一、windows下安装NPM

1.下载node.js的windows版

去 http://www.nodejs.org/ 下载最新版node.exe，并将它放到D:\nodejs目录下

2.设置环境变量

把D:\nodejs加入到PATH环境变量中。
配置成功后可以在CMD中通过node --version 看到node.js对应的版本号

C:\Users\Administrator>node --version
v0.10.30

3.安装 git

CMD中运行 git --version 确认安装是否成功

D:\Documents\GitHub> git --version
git version 1.8.4.msysgit.0

4.安装 npm

在确保node.exe和git都在PATH环境变量中后，可以在任意目录执行以下命令

git clone --recursive git://github.com/isaacs/npm.git
cd npm
node cli.js install npm -gf

CMD中运行 npm --version 确认安装是否成功

D:\Documents\GitHub\npm [master]> npm --version
1.3.21

安装成功后可将git克隆的npm目录删除，此时npm相关代码已经被安装在D:\nodejs目录中

二、安装 spm 工具

$ npm install spm@2.x -g
#打包
$ npm install spm-build -g

#查看版本号
$ spm --version
2.2.4

现在可以使用了

#打开js目录
$ cd path/to/js

#从默认源获取并安装指定版本seajs模块
$ spm install seajs@2.1.1
$ spm install crossjs/jquery@1.11.0

#打包所有js模块，自动创建dist目录并将打包后的js输出到该目录中
$ spm build

#安装模块自身到指定目录
$ spm install . -d sea-modules

三、安装 less 工具

$ npm install -g less
$ lessc --version
lessc 1.5.1 (LESS Compiler) [JavaScript]

接下来就可以使用lessc来编译.less文件了

$ lessc -x less/example.less css/example.css

大功告成！^o^
