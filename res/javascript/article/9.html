JS页面跳转

当前窗口跳转

window.location.href = 'page.html';  //方法1
window.history.back(-1);  //方法2，返回前一页面
window.navigate('page.html');  //方法3，针对IE的方法，其它浏览器不兼容
self.location = 'page.html';  //方法4

父窗口跳转 (跳出frame)

top.location = 'page.html';

扩展：判断当前页面是否被嵌入iframe内

if(top.location != self.location) {
alert(true);  //iframe内的页面
} else {
alert(false); //顶级窗口、弹出窗口
}
