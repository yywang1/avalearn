jQuery对象与DOM对象的转换

两种类型对象写入字符串测试

<div id="testa"></div>
<div id="testb"></div>
<div id="testc"></div>
<div id="testd"></div>

<script>
    var a = document.getElementById('testa'); //DOM对象
    a.innerHTML = 'aaa'; //成功

    var b = document.getElementById('testb'); //DOM对象
    b.html('bbb');       //失败，报错：Uncaught TypeError: Object #<HTMLDivElement> has no method 'html'

    var $c = $('#testc'); //jQuery对象
    $c.innerHTML = 'ccc'; //失败，但是无报错

    var $d = $('#testd'); //jQuery对象
    $d.html('ddd');       //成功
</script>

DOM 转 jQuery

用 $() 把DOM对象包装起来，就可以获得一个jQuery对象了

var b = document.getElementById('testb');  //DOM对象
b.html('bbb');   //失败，报错：Uncaught TypeError: Object #<HTMLDivElement> has no method 'html'

var $b2 = $(b);   //DOM 转 jQuery
$b2.html('bbb');  //成功

jQuery 转 DOM

两种转换方式：$obj[index] 和 $obj.get(index)

var $c = $('#testc');      //jQuery对象
$c.innerHTML = 'ccc';  //失败，但是无报错

var c2 = $c[0];             //jQuery 转 DOM
c2.innerHTML = 'ccc';    //成功
var $c = $('#testc');      //jQuery对象
$c.innerHTML = 'ccc';  //失败，但是无报错

var c2 = $c.get(0);       //jQuery 转 DOM
c2.innerHTML = 'ccc';    //成功
