不带按钮的无缝滚动

效果展示

代码展示

HTML

<div class="scrollL">
    <div class="scrollIn">
        <ul class="ulCont">
            <li><a href="#"><img src="../../../examples/pic1.jpg" alt="示例图片" /></a></li>
            <li><a href="#"><img src="../../../examples/pic2.jpg" alt="示例图片" /></a></li>
        </ul>
        <ul class="ulCopy">
        </ul>
    </div>
</div>

以向左滚动为例，其它方向滚动只有最外层标签的class不同，其它标签完全相同

CSS

/*向左滚动*/
.scrollL { float:left; overflow:hidden; width:920px; height:134px; background:#ccc; border:solid 7px #397300; }
.scrollL .scrollIn { float:left; }
.scrollL .ulCont { float:left; }
.scrollL .ulCopy { float:left; }
.scrollL li { float:left; width:178px; padding:0 3px; }
.scrollL img { width:178px; height:134px; }

/*向上滚动*/
.scrollT { float:left; overflow:hidden; width:178px; height:420px; background:#ccc; border:solid 7px #397300; margin-top:15px; }
.scrollT li { height:134px; padding:3px 0; }
.scrollT img { width:178px; height:134px; }

以向左和向上滚动为例，向右与向左样式相同，向下与向上样式相同

JS

<script type="text/javascript" src="../../../jquery-1.11.2.min.js"></script>
<script type="text/javascript" src="scrollL.js"></script>
<script type="text/javascript">
    $(document).ready(function(){
        $(".scrollL").scrollL({unitS:184,speed:10});    //unitS: 每个li所占的宽度或高度  speed: 滚动速度
    })
</script>

以向左滚动为例，其它基本相同

要点总结

  • 以上滚动js代码的原理是：将ulCont中的代码复制到ulCopy中，scrollIn开始滚动；当 ulCopy的最后一个标签完全进入可见区域时，scrollIn跳到ulCont 的最后一个标签完全进入可见区域的位置。
  • ulCopy的使用(实现无缝滚动)；
  • 滚动距离关键点的判断；
  • 头部调用函数时传参。
