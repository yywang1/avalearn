图片等比缩放(有加载中状态)

效果展示

按最大宽度和最大高度等比缩放，图片完整显示在该区域内
按最大宽度和最大高度等比缩放，图片不一定完整显示在该区域内，取中间部分显示
按最大宽度等比缩放，高度不限

代码展示

HTML

<ul class="imgList1">
    <li><img src="../../../examples/pic8.jpg" alt="示例图片" /></li>
    <li><img src="../../../examples/pic10.jpg" alt="示例图片" /></li>
    <li><img src="http://www.sxc.hu/pic/l/b/ba/ba1969/1380119_60107345.jpg" alt="示例图片" /></li>
    <li><img src="../../../examples/pic9.jpg" alt="示例图片" /></li>
</ul>

以第一块为例，其它结构与之相同

CSS

/*按最大宽度和最大高度等比缩放，图片完整显示在该区域内*/
.imgList1 { width:100%; overflow:hidden; float:left; padding-bottom:15px; }
.imgList1 li { width:200px; height:200px; overflow:hidden; float:left; border:solid 1px #ccc; margin-right:10px; text-align:center; }

/*按最大宽度和最大高度等比缩放，图片不一定完整显示在该区域内，取中间部分显示*/
.imgList2 { width:100%; overflow:hidden; float:left; padding-bottom:15px; }
.imgList2 li { width:200px; height:200px; overflow:hidden; float:left; border:solid 1px #ccc; margin-right:10px; }

/*按最大宽度等比缩放，高度不限*/
.imgList3 { width:100%; overflow:hidden; float:left; }
.imgList3 li { width:200px; overflow:hidden; float:left; border:solid 1px #ccc; margin-right:10px; text-align:center; }

JS

<script type="text/javascript" src="../../../jquery-1.11.2.min.js"></script>
<script type="text/javascript" src="imgSize.js"></script>
<script type="text/javascript">
    $(document).ready(function(){
        $(".imgList1 img").each(function(){ $(this).imgSize1({maxw:200,maxh:200}); });
        $(".imgList2 img").each(function(){ $(this).imgSize2({maxw:200,maxh:200}); });
        $(".imgList3 img").each(function(){ $(this).imgSize3({maxw:200}); });
    })
</script>
