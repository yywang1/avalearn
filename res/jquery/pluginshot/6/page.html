CSS

/*按钮*/
.btnLeft, .btnRight { display:block; width:16px; height:31px; overflow:hidden; text-decoration:-1000px; background:url(../../../images/btn.gif) no-repeat; }
.btnLeft { background-position:0 0; }
.btnRight { background-position:0 -33px; }
.btnTop, .btnBot { display:block; width:31px; height:16px; overflow:hidden; text-decoration:-1000px; background:url(../../../images/btn.gif) no-repeat; }
.btnTop { background-position:0 -66px; }
.btnBot { background-position:0 -84px; }

/*横向滚动*/
.scrollLR { background:#397300; padding:7px 33px; float:left; width:736px; position:relative; margin-bottom:10px; }
.scrollLR .btnLeft { position:absolute; top:59px; left:7px; }
.scrollLR .btnRight { position:absolute; top:59px; right:7px; }
.scrollLR .noAvail { filter:alpha(opacity=30); opacity: 0.3; cursor:default; }
.scrollLR .scrollWrap { float:left; overflow:hidden; width:736px; height:134px; background:#ccc; }
.scrollLR .scrollIn { float:left; }
.scrollLR .ulCont { float:left; }
.scrollLR .ulCopy { float:left; }
.scrollLR li { float:left; width:178px; padding:0 3px; }
.scrollLR img { width:178px; height:134px; vertical-align:bottom; }

/*纵向滚动*/

.scrollTB .btnBot { position:absolute; bottom:7px; left:74px; }

要点总结

  • 以上横向滚动1和纵向滚动1的js代码原理是：将ulCont中的代码复制到ulCopy中，scrollIn开始滚动；当 ulCopy的最后一个标签完全进入可见区域时，scrollIn跳到ulCont 的最后一个标签完全进入可见区域的位置。
  • ulCopy的使用；
  • 滚动距离关键点的判断。
