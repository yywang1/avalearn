代码展示

HTML

<div class="tab">
    <ul class="tabTit">
        <li>tab1</li>
        <li>tab2</li>
        <li>tab3</li>
    </ul>
    <div class="tabCont">
        <div class="tabCont_c">tab1的内容</div>
        <div class="tabCont_c">tab2的内容</div>
        <div class="tabCont_c">tab3的内容</div>
    </div>
</div>

CSS

.tab { width:400px; margin:auto; }
.tabTit { height:30px; overflow:hidden; background:url(../../../examples/line.gif) repeat-x 0 29px; border-left:solid 1px #285000; }
.tabTit li { float:left; height:28px; line-height:28px; overflow:hidden; cursor:pointer; padding:0 20px; background:#3e7b00; border-top:solid 1px #285000; border-right:solid 1px #285000; color:#fff; }
.tabTit .hov { background:#285000; }
.tabTit .cur { cursor:default; background:#fff; border-bottom:solid 1px #fff; color:#000; }
.tabCont { clear:both; border:solid 1px #285000; border-top:none; }
.tabCont_c { text-align:center; line-height:200px; }

JS

<script type="text/javascript" src="../../../jquery-1.11.2.min.js"></script>
<script type="text/javascript" src="funcTab.js"></script>
<script type="text/javascript">
$(document).ready(function(){
    $(".tab").each(function(){ $(this).funcTab(); });
});
</script>

相关文件下载

  • jquery基本包jquery-1.11.2.min.js
  • tab页封装文件funcTab.js

要点总结

  • 此段代码支持一页多个和多层嵌套；
  • 注意css中悬浮样式和选中样式的次序。
