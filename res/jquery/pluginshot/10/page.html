随机显示一张图片

效果展示

每次刷新页面，随机显示一张图片

代码展示

HTML

<ul class="randomUl">
    <li><img src="../../../examples/pic1.jpg" alt="示例图片" /></li>
    <li><img src="../../../examples/pic2.jpg" alt="示例图片" /></li>
    <li><img src="../../../examples/pic3.jpg" alt="示例图片" /></li>
</ul>

CSS

/*随机显示一张图片*/
.randomUl { width:600px; margin:auto; }
.randomUl li { display:none; }
.randomUl img { width:600px; }

JS

<script type="text/javascript" src="../../../jquery-1.11.2.min.js"></script>
<script type="text/javascript">
    $(document).ready(function(){
        var len=$(".randomUl").find("li").length;
        var i=Math.floor(Math.random()*len);    //随机生成一个0~len-1的整数
        $(".randomUl").find("li").eq(i).show();
    })
</script>

相关文件下载

  • jquery基本包jquery-1.11.2.min.js

要点总结

  • js随机函数 Math.random() 的使用。
  • Math.random() -- 返回0和1之间的伪随机数 可能为0，但总是小于1，[0,1)
