代码范例，鼠标悬浮时由100px变为200px


    






    
transition

    
transition: property duration timing-function delay;

    
transition 属性是一个简写属性，用于设置下文中的四个过渡属性：transition-property、transition-duration、transition-timing-function、transition-delay。






    
transition-property 应用过渡效果的 CSS 属性的名称

    
transition-property: none|all|property;


    
all：默认值，所有属性都将获得过渡效果

    

        

        

            
transition-property: all;

            
宽度和背景色都有过渡效果。

        

    


    
property：指定应用过渡效果的 CSS 属性名称列表，列表以逗号分隔

    

        

        

            
transition-property: background;

            
只有背景色有过渡效果，宽度没有。

        

    

    

        

        

            
transition-property: background,width;

            
宽度和背景色都有过渡效果。

        

    






    
transition-duration 完成过渡效果需要花费的时间（以秒或毫秒计）

    
transition-duration: time;

    

        

        
transition-duration: 1s;

    

    

        

        
transition-duration: 200ms;

    






    
transition-timing-function 过渡效果的速度曲线

    
transition-timing-function: linear|ease|ease-in|ease-out|ease-in-out|cubic-bezier(n,n,n,n);


    
linear：以相同速度开始至结束，cubic-bezier(0,0,1,1)

    

        

        
transition-timing-function: linear;

    


    
ease：慢速开始，然后变快，然后慢速结束，cubic-bezier(0.25,0.1,0.25,1)

    

        

        
transition-timing-function: ease;

    


    
ease-in：以慢速开始，cubic-bezier(0.42,0,1,1)

    

        

        
transition-timing-function: ease-in;

    


    
ease-out：以慢速结束，cubic-bezier(0,0,0.58,1)

    

        

        
transition-timing-function: ease-out;

    


    
ease-in-out：以慢速开始和结束，cubic-bezier(0.42,0,0.58,1)

    

        

        
transition-timing-function: ease-in-out;

    


    
cubic-bezier(n,n,n,n)： 贝塞尔曲线

    

        

        

            
transition-timing-function: cubic-bezier(0.9,0.1,0.1,0.9);

            
在 cubic-bezier 函数中定义自己的值，可能的值是 0 至 1 之间的数值，四个值分别为下图中P1、P2两个点的坐标(P0、P3是默认的点，对应了[0,0], [1,1])。相关工具：贝塞尔曲线设计器

            

            
其它参数均可通过cubic-bezier实现：

                ease: cubic-bezier(0.25, 0.1, 0.25, 1.0)

                linear: cubic-bezier(0.0, 0.0, 1.0, 1.0)

                ease-in: cubic-bezier(0.42, 0, 1.0, 1.0)

                ease-out: cubic-bezier(0, 0, 0.58, 1.0)

                ease-in-out: cubic-bezier(0.42, 0, 0.58, 1.0)
            

        

    






    
transition-delay 规定过渡效果何时开始

    
transition-delay: time;

    
time 以秒或毫秒计，默认值为0


    

        

        

            
transition-delay: 0;

        

    

    

        

        

            
transition-delay: 1s;
