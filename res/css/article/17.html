CSS3学习笔记 - 字体

@font-face 语法

CSS3中新增的 @font-face，能够加载服务器端的字体文件，让客户端显示客户端所没有安装的字体。语法如下：

@font-face {
    font-family: <YourWebFontName>;
    src: <source> [<format>][,<source> [<format>]]*;
    [font-weight: <weight>];
    [font-style: <style>];
}
  1. YousourcerWebFontName：自定义的字体名称，最好是使用你下载的默认字体，用于font-family属性。如font-family:"YourWebFontName";
  2. src：字体文件存放的先对或绝对路径；
  3. format：字体的格式，主要用来帮助浏览器识别，其值主要有以下几种类型：truetype、 opentype、truetype-aat、embedded-opentype、avg等；
  4. font-weight：字体是否为粗体；
  5. font-style：字体的样式，可能的值为：normal、italic、oblique。

format 字体格式

TureTpe(.ttf)

.ttf字体是Windows和Mac的最常见的字体，是一种RAW格式，因此他不为网站优化,支持这种字体的浏览器有【IE9+,Firefox3.5+,Chrome4+,Safari3+,Opera10+,iOS Mobile Safari4.2+】。

OpenType(.otf)

.otf字体被认为是一种原始的字体格式，其内置在TureType的基础上，所以也提供了更多的功能,支持这种字体的浏览器有【Firefox3.5+,Chrome4.0+,Safari3.1+,Opera10.0+,iOS Mobile Safari4.2+】；

Web Open Font Format(.woff)

.woff字体是Web字体中最佳格式，他是一个开放的TrueType/OpenType的压缩版本，同时也支持元数据包的分离,支持这种字体的浏览器有【IE9+,Firefox3.5+,Chrome6+,Safari3.6+,Opera11.1+】；

Embedded Open Type(.eot)

.eot字体是IE专用字体，可以从TrueType创建此格式字体,支持这种字体的浏览器有【IE4+】；

SVG(.svg)

.svg字体是基于SVG字体渲染的一种格式,支持这种字体的浏览器有【Chrome4+,Safari3.1+,Opera10.0+,iOS Mobile Safari3.2+】。

多浏览器兼容

推荐写法：

@font-face {
    font-family: 'YourWebFontName';
    src: url('../font/YourWebFontName.eot?') format('eot'),
         url('../font/YourWebFontName.woff') format('woff'),
         url('../font/YourWebFontName.ttf') format('truetype');
}

以上写法来自：Bulletproof @font-face Syntax

兼容更多浏览器的写法：

@font-face {
    font-family: 'YourWebFontName';
    src: url('../font/YourWebFontName.eot');
    src: url('../font/YourWebFontName.eot?#iefix') format('embedded-opentype'),
         url('../font/YourWebFontName.woff') format('woff'),
         url('../font/YourWebFontName.ttf') format('truetype'),
         url('../font/YourWebFontName.svg#NeuesBauenDemo') format('svg');
    font-weight: normal;
    font-style: normal;
}
