    display: block;

appearance 使元素看上去像标准的用户界面元素(浏览器默认的样式)，默认为normal

appearance: normal|icon|window|button|menu|field;

-webkit-appearance: button; -moz-appearance: button;

测试 button 元素



box-sizing 盒模型组成模式，默认为content-box

box-sizing: content-box|border-box|inherit;

box-sizing属性测试使用的样式，添加border和padding属性，宽高均为100px，背景为蓝色，content区域使用绿色填满

.box-sizing {
    border:solid 10px #000;
    padding: 10px;
}
.box-sizing:after {
    display: block;
    content: '';
    width: 100%;
    height: 100%;
    background: #7fbf4d;
}
box-sizing: content-box;

width和height不包含padding和border，内容区域宽高为100px，为默认样式。

box-sizing: border-box;

width和height包含padding和border，内容区域宽高为60px

outline-offset 对轮廓进行偏移，并在边框边缘进行绘制

outline-offset: length|inherit;
border:3px solid #000;
outline:3px solid #f00;
outline-offset: 0;

outline-offset需要outline属性一起设置

Ava

上面的容器：

border:3px solid #000;
outline:3px solid #f00;
outline-offset: 15px;

下面的容器：

border:3px solid #000;
outline:3px solid #f00;
outline-offset: -15px;

从左侧两个挨在一起的容器可以看出，轮廓不占用空间
