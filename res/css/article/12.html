transform 向元素应用2D或3D转换

transform: none|transform-functions;

none：不进行转换

transform: none;

translate(x,y)：根据给定的 left（x 坐标） 和 top（y 坐标） 位置参数，从当前位置移动

transform: translate(10px, 10px);

translateX(x)：沿X轴移动

transform: translateX(10px);

translateY(y)：沿Y轴移动

transform: translateY(10px);

translateZ(z)：3D 转换，沿Z轴移动

transform: translateZ(100px);

translateZ和translate3d都属于3D变换，需要跟perspective结合使用才能达到透视效果，父元素添加属性：
-webkit-perspective: 500;

transform: translateZ(-100px);

translate3d(x,y,z)：3D 转换

transform: translate3d(10px, 10px, -100px);

scale(x,y)

transform: scale(0.5, 1.5);

x, y为缩放的倍数

scaleX(x)

transform: scaleX(0.5);

scaleY(y)

transform: scaleY(0.5);

rotate(angle)：2D旋转

transform: rotate(45deg);

在参数中规定角度，角度单位是deg(度)

rotateX(angle)：沿着 X 轴的3D旋转

transform: rotateX(45deg);

rotateY(angle)：沿着 Y 轴的3D旋转

transform: rotateY(45deg);

rotateZ(angle)：沿着 Z 轴的3D旋转

transform: rotateZ(45deg);

rotate3d(x,y,z,angle)：3D旋转

transform: rotate3d(1,1,1,45deg);

rotate3d是rotateX, rotateY, rotateZ的综合，rotate3d(1,0,0,45deg)就相当于rotateX(45deg)。

skew(x-angle,y-angle)：2D 斜切

transform: skew(20deg, 20deg);

在参数中规定角度，角度单位是deg(度)，skew包含了skewX和skewY。

skewX(angle)

transform: skewX(20deg);

skewY(angle)

transform: skewY(20deg);

transform-origin 改变元素转换的基点

perspective 透视

perspective 属性定义了3D元素距视图的距离，单位为像素。

perspective 属性有两种写法：

写法一，用在舞台元素上（动画元素们的共同父辈元素），当为舞台元素定义perspective属性时，其子元素会获得透视效果，而不是该元素自身。

div {
    perspective: 500;
    -webkit-perspective: 500; /* Safari 和 Chrome */
}

写法二，用在当前动画元素上，与transform的其他属性写在一起。

transform: perspective(600px) rotateY(45deg);
