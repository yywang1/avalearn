border-radius 圆角

写法一

top-left
border-top-left-radius: 30px 15px;
top-left
border-top-left-radius: 30px;
top-right
bottom-right
bottom-left
左侧赋值均为 30px 15px，上下在前，左右在后。

写法二（简写）

border-radius: 10px 20px 30px 40px;

从左到右四个值对应的圆角分别是：左上、右上、右下、左下。

border-radius: 10px 20px 40px;

从左到右三个值对应的圆角分别是：左上、右上/左下、右下。

border-radius: 10px 40px;

从左到右两个值对应的圆角分别是：左上/右下、右上/左下。

border-radius: 10px;

四个角，不解释。

写法三：border-radius: 1-4 length|% / 1-4 length|%

border-radius: 10px / 20px;

“/”左侧为上下弧度，右侧为左右弧度，顺序参考写法二

box-shadow 阴影

box-shadow: h-shadow v-shadow blur spread color inset;

h-shadow: 必需，水平阴影的位置。

box-shadow: 5px 5px;
box-shadow: 15px 5px;

v-shadow: 必需，垂直阴影的位置。

box-shadow: 5px 5px;
box-shadow: 5px 15px;

blur: 模糊距离。

box-shadow: 0 0 5px;

上下左右4个方向各扩展了5px的模糊阴影

box-shadow: 5px 5px 5px;

上下左右4个方向各扩展了5px的模糊阴影后向右下方偏移5px

box-shadow: 5px 5px 10px;

spread: 阴影的尺寸。

box-shadow: 5px 5px 0 0;

spread为0时，阴影宽高与容器宽高（包含边框）相等

box-shadow: 0 0 0 5px;

四个方向黑色区域宽高均为5px，阴影整体宽高各增加10px

box-shadow: 5px 5px 0 5px;

阴影增加宽高10px后向右下方偏移5px

box-shadow: 0 0 5px 5px;

blur+spread测试，四个方向的投影各为10px，且10px区域都是模糊的，并不是5px实体+5px模糊区域

color: 阴影的颜色。

box-shadow: 5px 5px 0 0 #d00;

inset: 外部阴影 (outset) 改为内部阴影。

box-shadow: 0 0 0 0 #000 inset;
box-shadow: 0 0 0 5px #000 inset;
box-shadow: 0 0 5px 0 #000 inset;
box-shadow: 5px 5px 5px 5px #000 inset;

border-image 边框图片

border-image: border-image-source border-image-slice border-image-repeat;

边框背景图片(80%透明度); 每个色块尺寸为30px*30px

width: 100px; height: 60px; border: solid 15px #bbb;

该代码为下面容器的通用样式

border-image-source: 用在边框的图片的路径

border-image:url('{{res_path}}images/8_2.png');

默认剪裁位置100%

border-image-slice: 图片剪裁位置

border-image:url('{{res_path}}images/8_2.png') 30 round;

注意：border-image-slice没有单位，专指像素；
可以使用百分比，如：border-image:url('{{res_path}}images/8_2.png') 33.33% round;

切割原理，以 border-image-slice 为 10 20 30 40 为例。
按照左侧图片所示将边框背景图片切割为九个区域。
效果如下：

border-image:url('{{res_path}}images/8_2.png') 10 20 30 40 round;

border-image-repeat: 边框中间部分的重复类型

该属性有三个值：round（平铺）、repeat（重复）、stretch（拉伸）。

border-image:url('{{res_path}}images/8_2.png') 30 round;

round 背景重复，改变图片大小其正好在区域内显示

border-image:url('{{res_path}}images/8_2.png') 30 repeat;

repeat 背景重复，不改变图片大小，边缘会截断

border-image:url('{{res_path}}images/8_2.png') 30 stretch;

stretch 背景不重复，拉伸图片，充满整个区域

border-image-repeat 参数可以是 0~2个

border-image:url('{{res_path}}images/8_2.png') 30;

0个参数，即默认，效果为stretch

border-image:url('{{res_path}}images/8_2.png') 30 round;

1个参数，水平及垂直方向均使用此参数

border-image:url('{{res_path}}images/8_2.png') 30 round;

2个参数，水平方向使用第一个参数，垂直方向使用第二个参数

深入理解border-image的宽度和展示方式

border-image-slice 将边框背景图片切割成九个图片部分，包含左上、右上、右下、左下4个角边框图片和上右下左4个边框图片。

将容器边框也分为9个部分，与border-image一一对应，背景图片将分别填充进容器边框的相应区域。

其中4个角边框图片缩放后充满边框的4个角，类似stretch效果。

4个边框图片将按照border-image-repeat属性填充容器边框的相应区域。
