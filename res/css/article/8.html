border-radius 圆角

写法一

top-left
border-top-left-radius: 30px 15px;
top-left
border-top-left-radius: 30px;
top-right
bottom-right
bottom-left
左侧赋值均为 30px 15px

写法二（简写）

border-radius: 10px 20px 30px 40px;

从左到右四个值对应的圆角分别是：左上、右上、右下、左下。

border-radius: 10px 20px 40px;

从左到右三个值对应的圆角分别是：左上、右上/左下、右下。

border-radius: 10px 40px;

从左到右两个值对应的圆角分别是：左上/右下、右上/左下。

border-radius: 10px;

四个角，不解释。

写法三：border-radius: 1-4 length|% / 1-4 length|%

border-radius: 10px / 20px;

“/”左侧为上下弧度，右侧为左右弧度，顺序参考写法二

box-shadow 阴影

box-shadow: h-shadow v-shadow blur spread color inset;

h-shadow: 必需，水平阴影的位置。

box-shadow: 5px 5px;
box-shadow: 15px 5px;

v-shadow: 必需，垂直阴影的位置。

box-shadow: 5px 5px;
box-shadow: 5px 15px;

blur: 模糊距离。

box-shadow: 0 0 5px;

上下左右4个方向各扩展了5px的模糊阴影

box-shadow: 5px 5px 5px;

上下左右4个方向各扩展了5px的模糊阴影后向右下方偏移5px

box-shadow: 5px 5px 10px;

spread: 阴影的尺寸。

box-shadow: 5px 5px 0 0;

spread为0时，阴影宽高与容器宽高（包含边框）相等

box-shadow: 0 0 0 5px;

四个方向黑色区域宽高均为5px，阴影整体宽高各增加10px

box-shadow: 5px 5px 0 5px;

阴影增加宽高10px后向右下方偏移5px

box-shadow: 0 0 5px 5px;

blur+spread测试，四个方向的投影各为10px，且10px区域都是模糊的，并不是5px实体+5px模糊区域

color: 阴影的颜色。

box-shadow: 5px 5px 0 0 #d00;

inset: 外部阴影 (outset) 改为内部阴影。

box-shadow: 0 0 0 0 #000 inset;
box-shadow: 0 0 0 5px #000 inset;
box-shadow: 0 0 5px 0 #000 inset;
box-shadow: 5px 5px 5px 5px #000 inset;
