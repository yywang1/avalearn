分享到Facebook

<a href="http://www.facebook.com/sharer.php?u={$socialShareUrl}" target="_blank" rel="nofollow">Facebook</a>

Facebook分享按钮代码如下（放到服务器上才可用）

<fb:like href="http://developers.facebook.com/docs/reference/plugins/like" send="false" layout="button_count" width="450" show_faces="false"></fb:like>
<div id="fb-root"></div>
<script>
(function(d, s, id) {
var js, fjs = d.getElementsByTagName(s)[0];
if (d.getElementById(id)) return;
js = d.createElement(s); js.id = id;
js.src = "//connect.facebook.net/en_US/all.js#xfbml=1";
fjs.parentNode.insertBefore(js, fjs);
}(document, 'script', 'facebook-jssdk'));
</script>
